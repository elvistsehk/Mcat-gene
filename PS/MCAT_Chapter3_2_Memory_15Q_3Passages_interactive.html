🧠 MCAT Practice — Chapter 3.2 Memory (15Q + 3 Passages × 3Q)
📝 15 Standalone Questions
1. A student wants to remember a list of MCAT endocrine hormones by building a story that places each hormone into a meaningful medical scenario. Which encoding mode is the student using most strongly?
A. Semantic encoding
B. Visual encoding
C. Acoustic encoding
D. Automatic processing
✔️Correct!
❌Incorrect
Explanation (click to expand/collapse)
The student is putting information into a meaningful context, which is semantic encoding and is described as the strongest mode of encoding in this section.
  • B (Visual) is weaker and focuses on mental images without necessarily adding meaning.
  • C (Acoustic) stores the way something sounds (e.g., a rhyme), not a meaning-rich scenario.
  • D (Automatic processing) is passive and unintentional; building a story is effortful/controlled.
2. Order the following encoding modes from strongest to weakest, as described in the chapter: semantic, elaborative, acoustic, visual.
A. Visual > acoustic > elaborative > semantic
B. Semantic > elaborative > acoustic > visual
C. Elaborative > semantic > visual > acoustic
D. Acoustic > visual > semantic > elaborative
✔️Correct!
❌Incorrect
Explanation (click to expand/collapse)
The chapter states semantic encoding is the strongest and visual encoding is the weakest. Elaborative encoding (linking to existing knowledge) is deeper than acoustic, so the best order is: Semantic > Elaborative > Acoustic > Visual.
  • A reverses the strength ranking.
  • C incorrectly places visual above acoustic and swaps semantic/elaborative.
  • D places acoustic/visual above meaning-based encoding, contradicting the chapter’s hierarchy.
3. A student repeats “UAA, UGA, UAG” out loud over and over to keep the stop codons in mind during a short quiz. This is best described as:
A. Elaborative rehearsal
B. The self-reference effect
C. Maintenance rehearsal
D. Spreading activation
✔️Correct!
❌Incorrect
Explanation (click to expand/collapse)
Maintenance rehearsal is repetition used to keep information in consciousness/working memory or to help it progress toward longer-term storage.
  • A (Elaborative rehearsal) would connect stop codons to prior knowledge (e.g., “U = ‘You stop’”), not just repeat them.
  • B (Self-reference effect) is remembering better when information is linked to your own life.
  • D (Spreading activation) is activation of nearby nodes in a semantic network, not simple repetition.
4. Which example best illustrates elaborative rehearsal rather than maintenance rehearsal?
A. Repeating a phone number until you dial it
B. Whispering a definition repeatedly so you do not forget it
C. Reading the same paragraph again and again without adding meaning
D. Linking a new concept to prior knowledge by generating an analogy and a personal example
✔️Correct!
❌Incorrect
Explanation (click to expand/collapse)
Elaborative rehearsal means associating new information with knowledge already stored in long-term memory (often enhanced by the self-reference effect). Option D explicitly does this.
  • A, B, C are all forms of repetition without deep linking—classic maintenance rehearsal.
5. A student takes an exam in the same classroom where they studied and performs better than when testing in a different room. This pattern is best explained by:
A. The context effect (context-dependent memory)
B. The serial-position effect
C. Proactive interference
D. Decay of memory traces
✔️Correct!
❌Incorrect
Explanation (click to expand/collapse)
Context effects occur when memory is aided by being in the same physical location where encoding occurred—environmental cues help retrieval.
  • B concerns first/last items in a list (primacy/recency), not location.
  • C is old information interfering with new learning, not a location cue.
  • D is forgetting over time due to fading traces; the scenario is cue-dependent, not time-dependent.
6. Participants learn facts while intoxicated and later show better recall when intoxicated again than when sober. This is an example of:
A. The spacing effect
B. State-dependent memory
C. Source-monitoring error
D. Negative priming
✔️Correct!
❌Incorrect
Explanation (click to expand/collapse)
State-dependent memory is improved retrieval when a person is in the same mental/physiologic state as during encoding (the chapter explicitly uses intoxication as an example).
  • A is improved retention when study sessions are spaced apart.
  • C is confusing the origin/source of a memory.
  • D is when a prime interferes with processing of a later stimulus (slower responses/more errors), not state matching.
7. In list-learning experiments, participants recall the first and last items best immediately after learning. This phenomenon is known as:
A. The misinformation effect
B. The spacing effect
C. The serial-position effect (primacy + recency)
D. Retroactive interference
✔️Correct!
❌Incorrect
Explanation (click to expand/collapse)
The serial-position effect is higher recall for items at the beginning (primacy) and end (recency) of a list.
  • A is memory distortion due to outside misinformation.
  • B is about spacing study sessions over time.
  • D is new information interfering with old information.
8. A person has trouble recalling their new address because their old address keeps “popping in” first. This is best described as:
A. Retroactive interference
B. Decay
C. Confabulation
D. Proactive interference
✔️Correct!
❌Incorrect
Explanation (click to expand/collapse)
Proactive interference is when old information interferes with new learning (old address blocks the new one).
  • A (Retroactive) is the opposite direction: new information disrupts old memory.
  • B is loss over time due to fading traces; here the problem is competing similar information.
  • C is fabrication of memories (often linked to Korsakoff’s), not simple retrieval competition.
9. Which description best matches sensory memory?
A. A very brief, high-accuracy store that holds raw sensory input (e.g., iconic or echoic) for < 1 second
B. A store that holds ~7 ± 2 items for ~30 seconds and depends on consolidation for long-term storage
C. A system for manipulating information using the central executive, phonological loop, and visuospatial sketchpad
D. A virtually limitless warehouse of knowledge controlled entirely by the cerebral cortex
✔️Correct!
❌Incorrect
Explanation (click to expand/collapse)
Sensory memory preserves information in its original sensory form with high accuracy and lasts less than about a second (iconic = visual; echoic = auditory).
  • B describes short-term memory (capacity ~7 ± 2; duration ~30 s without rehearsal).
  • C describes working memory and the Baddeley-Hitch model components.
  • D describes long-term memory, but the chapter also notes hippocampal involvement in consolidation.

C. Procedural memory
D. Flashbulb memory
✔️Correct!
❌Incorrect
Explanation (click to expand/collapse)
The chapter characterizes short-term memory by a duration of about 30 seconds without rehearsal and a capacity around 7 ± 2.
  • A is a subtype of sensory memory for auditory input.
  • C is an implicit long-term memory for skills/habits.
  • D is a vivid explicit memory around emotionally important events.
11. Which option best describes working memory as presented in the chapter?

❌Incorrect
Explanation (click to expand/collapse)
Working memory lets you hold a few pieces of information and manipulate them (e.g., doing math in your head). The chapter ties it to short-term memory plus attention and executive function (frontal/parietal involvement).
  • A is iconic (sensory) memory.
  • B describes short-term memory but misses “manipulation.”
  • D is implicit long-term memory (procedural/conditioning).
12. According to the chapter, which working memory component supports mentally fitting puzzle pieces together by storing and manipulating visual/spatial information?

13. A patient cannot consciously recall having learned to ride a bicycle, but when handed a bicycle, rides smoothly. Which memory type best explains the preserved ability?

14. A word (“doctor”) is flashed briefly before a second word (“nurse”). Participants recognize “nurse” faster after “doctor” than after an unrelated prime. This is best described as:
A. Negative priming
B. Positive priming
C. Source amnesia
D. Retroactive interference
✔️Correct!
❌Incorrect
Explanation (click to expand/collapse)
Positive priming occurs when exposure to the first stimulus improves processing of the second, often measured by decreased response time. “Doctor” activates nearby semantic-network nodes (spreading activation), helping “nurse.”
  • A would slow responses or increase errors.
  • C is forgetting the source of information, not faster recognition.
  • D is new information interfering with old information, not a cueing benefit.

📘 Passage 1 — Study Context, Mood, and Retrieval Cues

Two friends, Maya and Luis, are preparing for a high-stakes exam. Maya studies in the same quiet library cubicle every day. She always uses the same peppermint gum and listens to a low-volume playlist with the same instrumental tracks. Luis studies in many places: sometimes on a noisy train, sometimes in a coffee shop, and sometimes at home with the television on. He also changes his routine frequently—different snacks, different music, and studying at different times of day.

On exam day, both students feel unusually anxious. During the first section, Maya notices the smell of peppermint from her own gum and begins to feel slightly calmer. She reports that certain definitions “pop into mind” as soon as she sees familiar keywords. Luis, in contrast, reports that he recognizes many topics but struggles to pull out precise details quickly, even though he remembers “seeing it somewhere.”

Afterward, the instructor comments that memory is helped when retrieval cues match the conditions present at encoding, and that some cues can be internal (like state and emotion) while others are external (like location and sensory details). The instructor also notes that recognition is generally easier than free recall, which is one reason multiple-choice exams can feel more manageable than fill-in-the-blank tests.

How to reason (Passage 1):
  1. Identify the manipulated variables: Maya keeps the same place and sensory cues (peppermint, playlist); Luis varies them.
  2. Translate cues into memory terms: Same place/sensory details → context cues. Internal anxiety/calmness → state/mood cues.
  3. Ask what the question is really testing: Is it about retrieval (recall vs recognition, cues), or about storage (STM/LTM) or distortion (misinformation)?
  4. Match evidence to the best mechanism: “Details pop in when seeing keywords” suggests cue-triggered retrieval (semantic network/spreading activation); stable study context predicts stronger context-dependent retrieval.
  5. Eliminate distractors: Interference/decay require competing lists or time-based fading evidence; those aren’t central here.
Tip: For MCAT passages, convert story elements (smell, room, mood) into named effects (context, state, priming), then choose the option that best fits the direction of the effect.
16. Which explanation best accounts for Maya’s improved recall when chewing peppermint gum during the exam?
A. Retroactive interference from new material learned on exam day
B. Decay of recent memories due to insufficient consolidation
C. Confabulation triggered by anxiety
D. A retrieval cue created by matching sensory context between encoding and retrieval
✔️Correct!
❌Incorrect
Explanation (click to expand/collapse)
The peppermint gum is an external sensory cue that was consistently present during studying. Having it present again during testing supports context-dependent retrieval (a context effect), where matching cues help pull memories out.
  • A requires evidence that new learning is disrupting old memories; the gum effect is cue-based, not competition-based.
  • B would predict worse recall over time regardless of cues; here recall improves with a matched cue.
  • C is vivid fabrication seen in specific disorders (e.g., Korsakoff’s), not a typical cueing effect.

📘 Passage 2 — Brain Injury, Consolidation, and Memory Systems

After a bicycle accident, a 31-year-old patient is evaluated for memory changes. The patient easily recalls childhood facts such as their birthdate and the names of elementary school teachers. However, they repeatedly forget that the clinician visited earlier that morning and ask the same introductory questions every few minutes. When presented with a new list of words, the patient cannot recall the list after a short delay unless they keep repeating it aloud continuously.

During occupational therapy, the patient is taught a new sequence of hand movements needed for a routine task. Over several days, the patient’s performance improves steadily, even though the patient insists each session feels “brand new” and cannot describe when or how they learned the sequence. The therapist also notes that when a specific cue word is shown at the start of practice, the patient performs the sequence faster than when an unrelated cue is shown.

Neuroimaging suggests injury affecting a structure important for consolidating short-term memories into long-term memory. The clinician explains that some types of long-term memory can remain relatively intact even when conscious recall of recent events is severely impaired.

How to reason (Passage 2):
  1. Separate “can’t form new memories” vs “lost old memories”: repeating questions & forgetting visits suggests anterograde issues (new memory formation).
  2. Map behaviors to memory types: improving a skill without conscious recall → implicit/procedural. Forgetting the visit → explicit episodic impairment.
  3. Use storage clues: needs constant repetition to hold a word list → short-term duration and maintenance rehearsal.
  4. Use brain-region clue: “structure important for consolidation” in this chapter points to the hippocampus.
  5. Use cue effects: a cue word improving performance → priming (and spreading activation).
Tip: In MCAT Psych/Soc passages, symptoms often encode the answer: “can do the skill but can’t explain it” nearly screams implicit/procedural.
19. Which conclusion is most supported by the patient improving on the hand-movement sequence while insisting each session is “brand new”?

B. Sensory memory is enhanced, allowing unusually long echoic traces
C. Procedural (implicit) memory is intact even when explicit recall of new experiences is impaired
D. The serial-position effect has been eliminated by the injury
✔️Correct!
❌Incorrect
Explanation (click to expand/collapse)
The passage provides a classic dissociation: the patient learns a skill (performance improves) but lacks conscious recollection of learning it. That pattern matches implicit/procedural memory preserved with impaired explicit episodic memory formation.
  • A does not fit: the patient recalls many facts (semantic) fine, but cannot recall recent episodes.
  • B is unrelated; sensory memory is seconds or less and does not explain multi-day skill learning.
  • D is about list position effects, not skill learning vs conscious recall.

21. The cue word that makes the patient perform the movement sequence faster is best explained by:
A. Priming that facilitates processing of a related stimulus

📘 Passage 3 — Eyewitness Memory, Misinformation, and Source Confusion

In a mock trial study, participants watch a short video of a car stopping at a yield sign. Later, half the participants read a written summary that accurately describes a yield sign, while the other half read a summary that incorrectly describes the car stopping at a stop sign. One week later, all participants are asked to describe the traffic sign they saw.

In a second phase, participants watch a different crash video. Some are asked, “How fast were the cars moving when they collided?” Others are asked, “How fast were the cars moving when they crashed?” Participants then estimate speed and later report whether they remember seeing broken glass—even though no glass was shown in the video.

Finally, researchers interview participants about a heroic “real-world” rescue story. The story details were actually taken from a movie, but several participants confidently report the story as if it were a factual news event. The researchers conclude that memory is not a perfect recording and that both outside information and internal reconstruction can distort recall.

How to reason (Passage 3):
  1. Spot the manipulation: the sign description is changed after viewing (yield vs stop) → outside info injected into memory.
  2. Look for leading language: “crashed” vs “collided” changes perceived severity → a classic leading-question manipulation.
  3. Identify the error type: reporting broken glass that wasn’t present is a false memory outcome driven by suggestion.
  4. Source clue: recalling a movie story as real is confusion about the origin/source of a memory.
  5. Choose the most specific label: “outside misinformation” → misinformation effect; “source of story” → source-monitoring error.
22. Participants who read the incorrect “stop sign” summary later insist they saw a stop sign in the video. This outcome most directly demonstrates:
A. Intrusion error caused by internally mixing two similar personal events
B. The misinformation effect from outside information injected into memory
C. Proactive interference caused by old knowledge of traffic rules
D. Agnosia preventing visual recognition of road signs
✔️Correct!
❌Incorrect
Explanation (click to expand/collapse)
The passage explicitly manipulates post-event information (the written summary) and shows it changes recall. That is the misinformation effect.
  • A (intrusion error) is internal mixing of similar memories, not an external summary replacing details.
  • C (proactive interference) would involve older learned info blocking new learning, not suggestion changing an observed detail.
  • D (agnosia) is an inability to recognize objects due to brain damage; the participants can recognize signs but recall the wrong one.
23. The finding that the word “crashed” increases speed estimates (and false reports like broken glass) is best explained by:
A. The spacing effect improving retention over one week
B. Negative priming interfering with later perception
C. Leading questions that bias memory reconstruction at recall
D. Eidetic memory persisting for minutes after viewing
✔️Correct!
❌Incorrect
Explanation (click to expand/collapse)
The only difference is wording (“crashed” vs “collided”), which acts as a leading question. This biases reconstructive memory, causing people to remember a more severe event and sometimes “remember” details that were never shown.
  • A relates to spaced learning sessions, not wording effects.
  • B would slow processing due to an interfering prime; it doesn’t explain inflated speed estimates and suggested details.
  • D is unrelated; eidetic memory is rare and would not be selectively changed by question wording.
24. Participants who describe a movie-based rescue story as a factual news event are most likely displaying:
A. Retroactive interference that erases the movie memory
B. Confabulation caused by thiamine deficiency
C. Intrusion error from mixing two personal New Year’s celebrations
D. Source-monitoring error confusing the origin of the memory
✔️Correct!
❌Incorrect
Explanation (click to expand/collapse)
The participants remember the content of the story but misattribute its source (movie vs real news). That is a classic source-monitoring error.
  • A (retroactive interference) would be new information disrupting old, but the key issue is misattribution of origin.
  • B (confabulation) is vivid fabrication typically associated with Korsakoff’s; here the details come from a real movie story, not invented filler.
  • C is an intrusion-error example, but the passage’s core mechanism is “where did I learn this?” which is source-monitoring.
