üìò Passage 3 - Visual Field Terminology, Optic Chiasm, and Reflexive Pathways

A neuroscience class demonstrates visual field mapping. A student fixates on a central point while an assistant briefly flashes a stimulus slightly to the student‚Äôs left. The instructor reminds the class that, because of geometry, objects in one side of visual space stimulate a specific side of the retina in each eye.

Next, the instructor introduces nasal versus temporal terminology for both visual fields and retinal fibers. For the student‚Äôs right eye, the flashed object is in the nasal visual field because photons must cross the nose to reach that eye. Those photons, however, strike the temporal retinal fibers of the right eye. For the left eye, the same object is in the temporal visual field and strikes nasal retinal fibers of the left eye.

How to reason:

  1. Visual geometry: each eye‚Äôs nasal visual field hits temporal retina, and temporal visual field hits nasal retina.
  2. At the optic chiasm, only nasal retinal fibers cross; temporal fibers stay ipsilateral.
  3. Conscious pathway goes through LGN ‚Üí visual cortex; reflexive eye movements/light reflexes use midbrain targets like superior colliculus.
  4. Pick the option that matches the specific step (field‚Üíretina mapping vs chiasm crossing vs reflex target).
